# KX7-STUDY
Physics AI Study Companion

## Features
- Chapter summaries
- AI Tutor for questions (LaTeX supported)
- Image Scanner
- Error Calculator
- Mobile friendly & Dark mode ready

## Setup
1. Clone repo
2. Run `npm install express node-fetch cors dotenv`
3. Create `.env` with `GEMINI_KEY=YOUR_KEY`
4. Run `node server.js`
5. Open `index.html` in browser
